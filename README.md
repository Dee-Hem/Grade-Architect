# 🎓 Grade Architect Pro

A Python-based academic management tool designed to help university students track their progress, calculate CGPA, and generate professional PDF reports.

## ✨ Key Features
* **Semester-Based Tracking:** Organize courses by academic level (e.g., 300L Harmattan).
* **Automated CGPA:** Real-time calculation using standard 5.0 scale logic.
* **Professional PDF Export:** Generate color-coded academic summaries instantly.
* **Offline-First:** No internet required; your data stays on your device.

## 🛠️ Technical Stack
* **Language:** Python 3.x
* **GUI:** Tkinter
* **PDF Generation:** FPDF library
* **Data Storage:** Local CSV (Excel-compatible)

## 🚀 How to Run
1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the script: `python.py`
